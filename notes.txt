- transition controla cómo será una transición, la misma se aplica al elemento (div)
- sirve para cualquier cambio de estado

- transition: propiedad + tiempo ("all" aplica a todos los cambios)

- se puede agregar timing-function

ease (default)
linear (mantiene la misma velocidad todo el tiempo) los cambios se hacen exactamente a la misma vel.
ease-in (empieza lento y va acelerando)
ease-out (empieza rápido y va más lento)

- skew inclina los elementos
- scale cambia o reduce el tamaño de un elemento (usa n° del 0 al 1 para achicar), scale 2 duplica el tamaño del elemento
- translate mueve los elementos (usar dos valores moverá el elemento tanto vertical como horizontalmente), valor de Y+ mueve las cosas para abajo, Y- las mueve para arriba
